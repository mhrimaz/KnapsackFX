About KnapsackFX

KnapsackFX is a JavaFX based program for solving Knapsack 0/1 problem with various algorithms like Dynamic Programming and Local Search algorithms like Hill Climbing (and its variations), Simulated Annealing, Genetic Algorithms, etc..
